# Microfrontend Rick and Morty

Creación de proyecto con Vite

### Para ejecutar en local el proyecto: 
```shell 
  npm run dev 
```

### Para compilar proyecto:
```shell 
  npm run build
```

### Para deployar, publicar:
```shell 
  npm run deploy
```
